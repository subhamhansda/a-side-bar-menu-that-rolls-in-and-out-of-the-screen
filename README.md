# a-side-bar-menu-that-rolls-in-and-out-of-the-screen
It's a side-bar menu that rolls in and out of the screen made with only CSS and HTML.
Functionality:
It rolls in over the screen when anyone clicks the hamburger icon and rolls out of the screen when the "X" button is pressed. 
***Several hovering effects are also been added to the icons ,and menu options.
